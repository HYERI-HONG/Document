Git Branch
===

> Branch

```bash
# git branch 목록
git branch

# 새로운 branch 생성
git branch "브랜치명"

# branch 전환
# branch 마지막 커밋 상태로 working directory 변경
git checkout "브랜치명"
```

> Merge

```bash
# 병합의 기준이 되는 branch로 cheakout
# 'b' --> 'a'
git checkout a

# branch 병합 
git merge b
```

- 다른 파일을 수정한 경우 -> 자동 병합
- 같은 파일, 다른 라인을 수정한 경우 -> 자동 병합
- 같은 파일, 같은 라인을 수정한 경우 -> 충돌(Conflict) 발생 -> 수동 병합

> Conflict

- 충돌 부분 수정 후 재 커밋
